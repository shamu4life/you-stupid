# you-stupid

A Cloudflare Worker prank: a fake "FBI domain seized" page that detonates into a youareanidiot.cc-inspired chaos engine on any interaction.

## Architecture

- `src/index.js` — Worker in front of every request (`assets.run_worker_first` in `wrangler.toml`). Serves `public/` via `env.ASSETS`; `GET /whoami` returns `{ip, city, region, country, isp, postal}` from `CF-Connecting-IP` + `request.cf`. Unknown HTML routes fall back to `/`. For HTML responses it runs an `HTMLRewriter` pass (`AbsoluteUrls`) that rewrites the page's root-relative `og:`/`twitter:` image + URL meta to absolute using the requested host — so the link-preview card resolves on `*.workers.dev` or any custom domain without hardcoding one.
- `public/index.html` — the entire experience. Generated by the build script (see below), not hand-edited.
- `public/media/voice1.mp3`, `voice2.mp3` — custom ElevenLabs voice clips. Silence-trimmed at build time by ffmpeg; runtime also trims via Web Audio.
- `public/images/emote.png` — custom Twitch emote (112×112) used everywhere the original had a smiley.
- `public/images/FBI_SEAL.png`, `DOJ_SEAL.png`, `background.png` — seized-page assets.
- `public/images/og.png` — the Open Graph / Twitter link-preview card (a crypto-casino promo). Referenced by the `og:`/`twitter:` meta in `index.html`; shown when the URL is shared, independent of the seizure page that actually loads.
- `wrangler.toml` — Workers + Static Assets config. `assets.run_worker_first = true` so the Worker runs ahead of asset serving and can rewrite the unfurl meta. Worker name: `you-stupid`.

## Build

`public/index.html` is generated by a Python script, not edited directly:

```bash
python3 /tmp/.../scratchpad/build_seized.py /tmp/.../scratchpad/full/index.html
```

The script lives in the session scratchpad. If it's gone (new session), reconstruct it from the current `public/index.html` — the HTML template is embedded verbatim in `TPL`. The script extracts two `<svg class="frame-black/white">` elements from the source HTML (vestigial; the build still expects them), inlines them as `__FRAME_BLACK__`/`__FRAME_WHITE__` placeholders, and writes `public/index.html`.

After editing the script, always syntax-check the output:

```bash
# extract inline JS and check
python3 -c "import re; open('/tmp/_c.js','w').write('\n;\n'.join(re.findall(r'<script[^>]*>(.*?)</script>', open('public/index.html').read(), re.S))); print('ok')"
node --check /tmp/_c.js
```

## Audio engine

Two clips loaded as separate `AudioBuffer`s. On every repeat, each playing "voice" flips a coin to pick clip 0 or 1 — so order is random (can repeat). Each voice is a self-perpetuating `chain(when)` call: plays a clip, schedules `chain` on `onended` after a 100ms breath. Stacked voices start staggered by up to 1.5s so they build into cacophony. Fallback: the `<audio id="snd">` element if Web Audio is unavailable.

## Key behaviours

- **Any gesture detonates** — `pointerdown` or `keydown` on the seized overlay hides it, swaps title/favicon, starts 2 voices and 6 popups (desktop) or 6 in-page sprites + vibrate (mobile).
- **Mobile detection** — UA string `/Android|iPhone|iPad|iPod/i` or iPadOS (`platform==="MacIntel" && maxTouchPoints>1`). Mobile gets bouncing `<div>` sprites instead of `window.open`.
- **Popup behaviour** — each popup click: 1 more voice + 3 more popups. Popups bounce via `window.moveTo` random-walk within `availLeft/Top` bounds.
- **Gating** — right-click block and anti-Alt+F4 alert only activate after the reveal. `onbeforeunload` is ungated (generic browser dialog).

## Local dev

```bash
npm install
npm run dev   # http://localhost:8787
```

`/whoami` returns `{}` fields locally (no real CF headers). Test geo display manually against the deployed preview URL.

## Deploy

Cloudflare Git integration auto-deploys `main` on push. Manual deploy: `npm run deploy`. Preview URLs appear as PR comments from the Cloudflare bot.

## Workflow

All changes go on a feature branch → PR → squash merge to `main`. The Cloudflare bot posts a deploy status comment on every PR; wait for ✅ before merging.
